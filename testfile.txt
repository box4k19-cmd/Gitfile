Git — это система контроля версий, то есть инструмент для отслеживания изменений в файлах и совместной работы над проектами.
Git помогает:сохранять историю изменений файлов, возвращаться к предыдущим версиям, работать над проектом нескольким людям одновременно, не мешая друг другу,безопасно экспериментировать с кодом.
С Git можно:

посмотреть, что именно изменилось;

откатиться к рабочей версии;

сохранить несколько вариантов решения.
